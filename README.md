IPL Auction Intelligence Dashboard (2013–2026)
Project Overview
This project analyzes IPL auction data from 2013–2026 using Python and Power BI. The goal is to identify:
•	undervalued players 
•	overvalued players 
•	team spending patterns 
•	auction trends 
•	role-based pricing insights

________________________________________
Key Insights
•	Punjab Kings spent the most money in auctions 
•	All-rounders had the highest average auction value 
•	Auction prices significantly increased after 2023 
•	Some players delivered high performance at lower prices
